このレッスンではマークダウンでリストを作ります。

この世界には二種類のリスト、番号なし(unordered)と番号付き(ordered)があります。 一つは箇条書きのリスト、もう一つは番号が付いているリストです。

番号なしリストをつくるには、リストの各項目をアスタリスク（　*　）で始めます。 また、各項目は一行ずつ記載します。例えば買い物リストをマークダウンで作るにはこのようにします。

* ミルク
* 卵
* 鮭
* バター

このマークダウンのリストは下のように表示されます。

  • 牛乳
  • 卵
  • 鮭
  • バター

この問題では、読点で区切られた単語をリスト形式にしてください。

できましたね！番号なしリストはこれでおしまいです。では、番号付きリストを見ていきましょう。

番号付きリストはアスタリスクのかわりに番号が最初につきます。こちらを見てください。

  1. ボウルに卵を３つ割り入れます。
  2. ボウルに牛乳を１ガロン入れます。
  3. 鮭にバターをしっかりと塗り込みます。
  4. 鮭を卵と牛乳が入っているボウルに入れます。

マークダウンでこれをやるには、下のようにします。

1. ボウルに卵を３つ割り入れます。
2. ボウルに牛乳を１ガロン入れます。
3. 鮭にバターをしっかりと塗り込みます。
4. 鮭を卵と牛乳が入っているボウルに入れます。

簡単ですね。リストそのままです。

この問題では、レシピの残りを番号付きリストにしてください。

素晴らしい出来です！

イタリックや太字、リンクをリストの中に入れることもできます。 この問題では、カッコ内の植物のラテン語名をイタリックにしてください。

最高です！

時には複数階層のリストやリストを別のリストの入れ子にしたい場合もあるでしょう。 心配ございません、マークダウンでも同じように書くことができます。 方法は、前の項目に対してスペースひとつ以上でインデント（字下げ）するだけです。

例えば、次のリストではメインのキャラクターにそれぞれの特徴を表すサブリストを加えています。

* タンタン
 * 記者
 * ボサっとしたオレンジの髪
 * 世界一の犬と友達
* ハドック
 * 船長
 * すばらしいひげの持ち主
 * ウイスキーが大好き
   * スコッチも？

これが表示される時には下のようになります。

  • タンタン
    • 記者
    • ボサっとしたオレンジの髪
    • 世界一の犬と友達
  • ハドック
    • 船長
    • 素晴らしいひげの持ち主
    • ウイスキーが大好き
      • スコッチも？

この問題では、キャラクターの特徴をサブリストにしてください。

すばらしい！入れ子にできるリストの数に制限はありませんが、３つまでにしておいた方が良いでしょう。 それ以上になると、複雑になりすぎてしまいます。

リストと字下げについて、段落を作るのに役立つことをもうひとつ覚えましょう。 箇条書きにリストではない補足情報をつける時には、このようにすることができます。

  1. ボウルに卵３つを割り入れます。

    汚さないように卵を割り入れてくださいね。

    もし汚してしまったら、しっかりときれいにしてください！

  2. ボウルに１ガロンの牛乳を注ぎます。

    前のステップと同じように、汚さないように気をつけてください。汚したらきれいにすること！

  3. 鮭にバターをしっかりと塗り込みます。

    「しっかり」というのは上下にという意味です。ジュリア・チャイルドはこのように言っています。

    上に、下に、そして全体に。バターを鮭に塗る時は忘れずに。

  4. 鮭を卵と牛乳が入っているボウルに入れます。

    鮭を落とすときのテクニックを紹介します。

    • 子供など邪魔がいないのを確認します
    • 両手を使います
    • 汚れたときのためにタオルを近くにおいておきます

このようにするには、それぞれの段落の前にひとつ以上の半角スペースがある必要があります。 例えば、上のリストはマークダウンで次のように書きます。

1. ボウルに卵３つを割り入れます。

 汚さないように卵を割り入れてくださいね。

2. ボウルに１ガロンの牛乳を注ぎます。

 前のステップと同じように、汚さないように気をつけてください。汚したらきれいにすること！

3. 鮭にバターをしっかりと塗り込みます。

   > 上に、下に、そして全体に。バターを鮭に塗る時は忘れずに。
4. 鮭を卵と牛乳が入っているボウルに入れます。

   鮭を落とすときのテクニックを紹介します。

   * 子供など邪魔がいないのを確認します

最初の二つは各項目の前に半角スペースが一個だけ入っています。見た目が よくないので、３番や４番のようにもっと半角スペースを入れても問題ありません。 この段落には、引用や他のリストなどのマークダウン要素を入れることもできますよ！

この問題では、箇条書きになっている部分をそれぞれの段落になるように変更してください。

これでリストの書き方は問題ありませんね！
