We beginnen met twee opmaakelementen. Schuingedrukt en vetgedrukt. In deze lessenserie zie je soms code in het rood. Doordat Markdown er niet anders uitziet dan 'gewone' tekst zie je het verschil niet. Door de code rood te maken zie je direct welke code zorgt voor de opmaak.

Om iets schuingedrukt te maken in Markdown dan gebruik je daarvoor het platliggende streepje (_ ). Voorbeeld, dit woord wordt schuingedrukt.

De eerste oefening, maak het woord "niet" schuingedrukt.

Goed gedaan! Dat was inderdaad het goede antwoord.

Zoals met schuingedrukt kun je ook tekst vetgedrukt maken. Hiervoor heb je twee sterretjes( ** ) nodig. Door iets **vetgedrukt** te maken kun je eenvoudig de aandacht richten op een woord of zin.

Oefening twee maak het woord make the word "will" bold.

Goed gedaan! Dat was inderdaad het goede antwoord.

Natuurlijk kun je de code voor schuingedrukt en vetgedrukt door elkaar gebruiken. Weet je nog welke code wat doet? Het sterretje * zorgt voor vetgedrukt en het laagliggende _streepje zorgt voor schuingedrukt Je kunt natuurlijk ook meerdere woorden achter elkaar opmaken.

Opdracht drie is, maak "Natuurlijk" schuingedrukt en het woord 'cola' vetgedrukt.

Fantastisch!

Dit is de laatste oefening in dit hoofdstuk. We gaan onze superpowers nu combineren Vetgedrukt en schuingedrukt doorelkaar gebruiken.

Het maakt niet uit waar je de sterretjes of laagliggende streepjes plaatst. In onderstaande voorbeeld maak het tweede gedeelte van de zin vetgedrukt en schuingedrukt. Maak "dan heb je waarschijnlijk ongelijk" zowel vetgedrukt als schuingedrukt. Begin met het sterretje aan de **_buitenkant_**

Je hebt twee nieuwe superpowers! Vetgedrukt en schuingedrukt in Markdown heb je nu onder de knie.
