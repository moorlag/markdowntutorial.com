Si vous savez comment créer des liens dans Markdown, vous pouvez aussi créer des images. Le la syntaxe est presque la même.

Les images ont aussi deux styles, tout comme les liens, les deux image ont exactement le même rendu. La différence entre les liens et les images est que les images sont précédés d'un point d'exclamation ( ! ).

Le premier style d'image s'appelle un lien d'image dans la ligne (inline image link) . Pour créer un fichier en ligne entrez un point d'exclamation ( ! ), enveloppez le texte alt entre parenthèses ( [ ] ), puis enveloppez le lien entre parenthèses ( ( )) ). (Le texte alternatif est une expression ou une phrase qui décrit l'attribut pour les malvoyants).

Par exemple, pour créer un lien image en ligne vers https://octodex.github.com/images/bannekat.png, avec un texte alt qui dit, Benjamin Bannekat, que vous ecrirez ça en Markdown : ![Benjamin Bannekat](https://octodex.github.com/images/bannekat.png).

Dans la case ci-dessous, dirigez le lien vers une image, et remplissez les crochets de texte alt. pour dire "Un joli tigre" :

Merveilleux !

Bien que vous n'ayez pas besoin d'ajouter du texte alt, cela rendra votre contenu plus accessible à votre public, y compris les personnes ayant une déficience visuelle, qui utilisent des lecteurs d'écran ou qui n'ont pas de connexion Internet haute vitesse.

Pour une image de référence, vous suivrez le même modèle qu'un lien de référence. Vous précéderez le markdown d'un point d'exclamation, puis fournissez deux crochets pour le texte alt, puis deux autres pour la balise image, comme ceci : [Le père fondateur][Pere] Au bas de votre démarque vous définirez une image pour la balise, comme ceci : [Pere] : http://octodex.github.com/images/founding-father.jpg.

Dans la boîte ci-dessous, nous avons commencé à placer quelques images de référence ; vous allez doivent les compléter, tout comme la dernière leçon. Appeler la première étiquette de référence "Noir", et construisez lui un lien vers https://upload.wikimedia.org/wikipedia/commons/a/a3/81_INF_DIV_SSI.jpg ; créez le second lien image vers http://icons.iconarchive.com/icons/google/noto-emoji-animals-nature/256/22221-cat-icon.png.

Ta da! Vous avez appris tout ce qu'il y a pour ajouter des images avec Markdown !
